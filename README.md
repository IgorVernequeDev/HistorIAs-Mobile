# 📖 HistorIAs Mobile

Aplicativo mobile que gera histórias automaticamente usando Inteligência Artificial.

O **HistorIAs Mobile** permite que o usuário crie histórias únicas escolhendo elementos como personagens, personalidade e gênero da narrativa. A partir dessas informações, a aplicação utiliza IA para gerar uma história completa.

## 🚀 Funcionalidades

* ✨ Geração automática de histórias com IA
* 👤 Escolha de personagens
* 🎭 Definição de personalidade
* 📚 Seleção de gênero da história
* 📱 Interface simples e intuitiva para mobile
* 🔄 Cada história gerada é única

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/IgorVernequeDev/HistorIAs-Mobile.git
```

Entre na pasta do projeto:

```bash
cd HistorIAs-Mobile
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npx expo start
```

Depois disso você pode abrir no:

* Expo Go (Android / iOS)
* Emulador Android
* Simulador iOS
